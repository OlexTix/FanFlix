## How To Use

1. Clone this repository into your folder
2. Go into the repository
```
cd FanFlix
```
3. Use this command to install all required dependencies:
```
npm install
```
4. Launch the project using this command:
```
npm run serve
```
5. Open your browser and type:
```
http://localhost:8080/
```
